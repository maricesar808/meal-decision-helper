# meal-decision-helper

Mini Project 2: Interactive Web Tool

## What It Does

Meal Decision Helper is a simple browser-based tool that helps choose where or what to eat. The user adds 2 to 10 meal or restaurant options, rates each one for cost, healthiness, and travel time, then calculates a ranked top 3 list.

The app also includes a Random Pick button that chooses one option from the calculated top choices.

## How To Use It

1. Open `index.html` in a browser.
2. Enter a meal or restaurant name.
3. Choose 1-5 ratings for cost, healthiness, and travel time.
4. Add at least 2 options.
5. Click Calculate to see the top 3 ranked choices.
6. Use Random Pick if you want the app to choose from the top results.
7. Use Edit, Delete, or Reset to change the list.

## Project Guidelines Checklist

- Semantic HTML structure: `index.html`
- Intentional CSS styling: `styles.css`
- Vanilla JavaScript only: `script.js`
- JavaScript state: meal options array and current ranking array
- Derived output: total scores, ranked top 3 list, random pick result
- At least 3 interaction types: form submit, text input, dropdown selection, button clicks
- Dynamic DOM updates: option list, rankings, errors, and random pick update without alerts
- Reset without refreshing: Reset button clears the current state
- Static and self-contained: no server, API, framework, library, or CDN
